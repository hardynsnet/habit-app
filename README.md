# habit-app
Aplicación para registro de hábitos personales

1. Objetivo del Software
   Permitir a los usuarios registrar hábitos personales.
   Llevar un conteo del cumplimiento de cada hábito.
   Mostrar métricas claras de progreso.
   Fomentar la constancia mediante estadísticas visuales.

2. Módulo de Usuario
  Funcionalidades:
   1. Registro de usuario
   2. Inicio de sesión
   3. Perfil básico del usuario
  
  Datos del usuario
   1. Nombre
   2. Correo electrónico
   3. foto de perfil

3. Módulo de Hábitos
  Funcionalidades:
   1. Crear hábitos
   2. Editar hábitos
   3. Eliminar hábitos
   4. Activar / desactivar hábitos
   5. Marcar hábito como cumplido
  Características del hábito:
   1. Nombre del hábito
   2. Frecuencia:
      1. Diario
      2. Semanal
      3. Mensual
  3. Fecha de creación
  4. Estado (activo / inactivo)
